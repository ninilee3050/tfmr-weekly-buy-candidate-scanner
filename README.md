# TFMR 1회차 매수 기준봉 검증 도구

이 프로그램은 TFMR 1회차 매수 기준봉 탐지 규칙을 Python 코드가 정확히 이해했는지 확인하기 위한 검증용 도구입니다. 자동매매 프로그램이 아니며, 주문 실행, 계좌 연결, 수익률 계산을 하지 않습니다.

프로그램 실행은 `실행하기.bat`을 더블클릭하면 된다.

## 처음 한 번 준비

Python이 설치되어 있어야 합니다. 필요한 패키지는 아래 명령으로 설치할 수 있습니다.

```powershell
pip install -r requirements.txt
```

## 화면 구성

화면은 3단 구조입니다.

- 왼쪽: 미국 시가총액 Top100 목록
- 중앙: 티커 검색창과 과거 1회차 매수 기준봉 표
- 오른쪽: Top100 이번주 매수후보 스캐너

왼쪽 Top100 표에서 종목을 클릭하면 중앙 검색창에 티커가 자동 입력되고 바로 검색됩니다.

## Top100 목록

Top100 목록은 StockAnalysis의 시가총액 순위 페이지를 실시간으로 조회해서 표시합니다.

실시간 조회에 실패하면 내장 목록으로 대체하지 않습니다. 이 경우 Top100 표는 비워두고, 인터넷 연결 또는 사이트 응답을 확인하라는 안내를 표시합니다.

`weekly_scan.py`도 Top100 실시간 조회에 실패하면 기본 목록으로 대체하지 않고 실패로 종료합니다.

## 중앙 티커 검색과 오른쪽 스캐너의 차이

중앙 티커 검색은 과거 확정 매수 기준봉을 보는 용도입니다. 검색할 때마다 Yahoo 계열 일봉 데이터를 새로 받아 `data/{TICKER}.csv`를 갱신하고, 진행 중인 이번 주 주봉은 제외한 뒤 확정 주봉 기준으로만 보여줍니다.

오른쪽 Top100 스캐너는 이번 주에 새로 발생한 후보를 찾는 용도입니다. 진행 중인 이번 주 주봉을 포함하지만, 이 결과는 확정 매수포인트가 아니라 `이번주 매수후보`입니다.

두 기능 모두 단순히 현재 주봉 조건만 필터링하지 않고, 과거 주봉을 처음부터 추적해서 같은 상승 사이클에서 이미 1회차 기준봉이 나왔는지 확인합니다.

## 데이터 기준

- Yahoo 계열 일봉 데이터를 사용합니다.
- Yahoo의 `1wk` 주봉을 그대로 쓰지 않고, 일봉 데이터를 월요일 시작 기준 주봉으로 직접 묶습니다.
- 주봉 날짜는 해당 주봉의 시작일인 월요일로 표시합니다.
- MA5, MA20, MA50, MA150, MA200은 모두 주봉 Close 기준 SMA입니다.
- EMA와 거래량 조건은 v1 매수 기준봉 조건에 사용하지 않습니다.

## TFMR v1 매수 기준봉 요약

1회차 기준봉은 아래 흐름을 만족하는 첫 음봉 주봉입니다.

- MA20이 MA50을 골든크로스해서 상승 사이클이 시작됨
- 상승 사이클 안에서 Close > MA20 이력이 있음
- 이후 Close < MA5 그리고 Close < MA20 눌림 상태가 발생함
- 그 눌림 안에서 Close < Open 음봉이 나옴
- MA150 > MA200 장기추세 정배열임
- 같은 상승 사이클에서 아직 1회차 기준봉이 기록되지 않음

모든 비교는 엄격 비교입니다. 같은 값은 조건 만족으로 보지 않습니다.

## v1에서 하지 않는 것

v1에서는 추가매수, 매도, 보유상태, 수익률 계산, 백테스팅, 브로커 주문 연동을 하지 않습니다. 먼저 1회차 매수 기준봉 탐지를 안정화하고, 향후 v2에서 추가매수와 매도 흐름 계산을 확장할 수 있습니다.

## 저장 파일

중앙 티커 검색은 검색할 때마다 아래 파일을 저장합니다.

```text
outputs/{TICKER}_tfmr_buy_points.csv
```

오른쪽 Top100 스캐너는 자동 저장하지 않습니다. 사용자가 `스캔 저장하기` 버튼을 눌렀을 때만 다운로드 폴더에 저장합니다.

```text
Downloads/top100_tfmr_candidates_YYYY-MM-DD.csv
Downloads/top100_tfmr_failures_YYYY-MM-DD.csv
```

명령형 스캔 파일 `weekly_scan.py`는 GUI 없이 Top100 이번주 후보 스캔을 실행합니다. GitHub Actions나 자동화 환경에서는 아래처럼 사용할 수 있습니다.

```powershell
python weekly_scan.py --limit 100 --output-dir outputs
```
